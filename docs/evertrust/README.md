## How to use this pack

- Read top-to-bottom. Each file builds on the previous one.
- Anything in **bold** is locked policy — don't relitigate without your team lead.
- Anything marked ❄ FROZEN or PARKED is intentionally out of scope right now. Don't try to fix it.
- You will hear words like *Argus*, *Hermes*, *Sieve*, *GAEB*, *Vergabe*, *T-2*, *LV*, *L5 PIC*. They're all in the glossary.

## After you finish reading

Talk to your team lead about:
- which **niche** (LED, Container, Cleaning) you'll start with
- which **phase** of the workflow you'll be working in
- whether you'll be touching the **ERP code**, the **n8n workflows**, or the **operational side** (running real tenders)
